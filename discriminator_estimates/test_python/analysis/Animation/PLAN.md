## 4. Архитектура (3 скрипта)

### 4.1. `anim_interactive.py` — Главный: интерактивное окно

**Формат запуска**: `python3 Animation/anim_interactive.py` (открывает окно)

**Макет окна** (≈ Simulink Scope):

```
┌───────────────────────────────────────────────────────────────┐
│                   ВЕРХНИЙ ГРАФИК (70%)                        │
│                                                               │
│   sinc(x - x0) — непрерывная кривая (голубая)                │
│   ● ● ● — 3 отсчёта на сетке (жёлтые точки)                 │
│   ┆ — истинный пик x0 (зелёный пунктир)                      │
│   │ — оценки: CG (красная), QA (бирюзовая),                  │
│       EA (жёлтая), AUTO (зелёная яркая, толстая)             │
│                                                               │
├───────────────────────────────────────────────────────────────┤
│                   НИЖНИЙ ГРАФИК (30%)                         │
│                                                               │
│   "Осциллограмма" ошибки:                                    │
│   История |err| последних 100 кадров для каждого метода       │
│   (скользящее окно, как в scope)                              │
│                                                               │
├───────────────────────────────────────────────────────────────┤
│  ПАНЕЛЬ УПРАВЛЕНИЯ                                            │
│                                                               │
│  [━━━━━━━━━●━━━━━━] x0 = 0.25  (положение пика)             │
│  [━━━●━━━━━━━━━━━━] SNR = 0.02 (уровень шума)               │
│  [━━━━━━━●━━━━━━━━] step = 1.0 (шаг сетки)                  │
│                                                               │
│  [▶ Play] [⏸ Pause] [⏺ Record GIF] [🔄 Reset]              │
│                                                               │
│  Текущие ошибки:                                              │
│  CG=0.234  SD=0.209  QA=0.007  EA=0.005  AUTO=0.005 (EA)    │
│  AUTO mode: EA ✓  │  Zone: normal  │  Monotonic: No          │
└───────────────────────────────────────────────────────────────┘
```

**Режимы работы**:

| Режим | Управление | Что происходит |
|-------|-----------|---------------|
| **Manual** | Слайдер x0 | Пользователь двигает пик вручную |
| **Play** (авто-sweep) | Кнопка ▶ | x0 автоматически плывёт от -1.5 до +1.5 |
| **Play + noise** | Кнопка ▶ + SNR > 0 | Sweep + случайный шум на каждом кадре |
| **Record** | Кнопка ⏺ | Play-режим записывается в GIF |

**Технические детали**:
- Бэкенд: `TkAgg` (работает на Debian)
- `FuncAnimation` с `blit=True` для плавности (перерисовка только изменившихся artists)
- `matplotlib.widgets.Slider` для x0, SNR, step
- `matplotlib.widgets.Button` для Play/Pause/Record/Reset
- Interval = 50ms (20 fps для интерактива)
- Нижний "scope" — ring buffer последних 100 значений ошибок

## 7. Ключевые технические решения

### 7.1. blit=True — критично для производительности

```python
# Вместо перерисовки ВСЕГО окна — перерисовываем только изменившиеся элементы
anim = FuncAnimation(fig, update, frames=301, blit=True, interval=50)

def update(frame):
    # Обновляем ТОЛЬКО данные, не пересоздаём artists
    line_sinc.set_ydata(sinc(x_fine - x0))
    dots.set_offsets(...)
    vline_true.set_xdata([x0])
    return [line_sinc, dots, vline_true, vline_cg, ...]
```

Без blit — ~5 fps. С blit — ~30+ fps.

### 7.2. Слайдеры НЕ совместимы с FuncAnimation.blit

**Проблема**: `matplotlib.widgets.Slider` + `FuncAnimation(blit=True)` конфликтуют.
Slider перерисовывает свою ось → blit ломается → мерцание.

**Решение**: Два режима:
- **Manual mode**: слайдеры без FuncAnimation, обновление через `slider.on_changed()`
- **Play mode**: FuncAnimation без слайдеров (слайдеры замораживаются), blit=True

### 7.3. Ring buffer для scope

```python
class RingBuffer:
    """Кольцевой буфер для "осциллографа" — хранит последние N значений."""
    def __init__(self, size=200):
        self.data = np.full(size, np.nan)
        self.idx = 0

    def push(self, value):
        self.data[self.idx % len(self.data)] = value
        self.idx += 1

    def get(self):
        """Возвращает данные в правильном порядке (старые слева)."""
        n = len(self.data)
        return np.roll(self.data, -self.idx % n)
```
