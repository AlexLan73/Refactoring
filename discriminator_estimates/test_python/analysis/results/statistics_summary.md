# Статистика точности дискриминаторов по зонам

## Что здесь измеряется?

Сдвигаем пик sinc(x) = sin(x)/x от x0 = −1.5 до +1.5 (301 точка).
На каждом шаге "снимаем" 3 отсчёта на сетке {−1, 0, +1} и пытаемся
**восстановить** положение пика по этим 3 точкам.

**5 методов**:
- **CG** — центр тяжести (2 точки, простой)
- **SD** — суммарно-разностный (2 точки, с коэффициентом c)
- **QA** — квадратичная аппроксимация (3 точки, парабола)
- **EA** — экспоненциальная аппроксимация (3 точки, Гауссиан)
- **AUTO** — автоматический: EA в normal зоне + E2 экстраполяция в extreme

**Метрики**: MAE = средняя ошибка, MaxErr = наихудшая, RMSE = среднеквадратичная,
Bias = систематическое смещение, Std = разброс, P95 = "в 95% случаев не хуже"

**Зоны**: normal (|x0|≤0.5, рабочий режим), boundary (0.5<|x0|≤1.0, пик у края),
extreme (|x0|>1.0, пик за сеткой), monotonic (нет перегиба)

## Краткое резюме

```
Рейтинг методов (от лучшего к худшему):

  Normal:     AUTO = EA (0.005) > QA (0.006) >> SD (0.205) > CG (0.226)
  Boundary:   AUTO = EA (0.043) > QA (0.070) >> SD (0.211) > CG (0.233)
  Extreme:    AUTO = E2 (0.256) > QA (0.585) ≈  SD (0.622) ≈ CG (0.688)
  Monotonic:  AUTO = E2 (0.148) > QA (0.324) >  SD (0.412) > CG (0.456)

  AUTO — лучший во ВСЕХ зонах!
```

**Рекомендация**: использовать `discr3_auto()` — один вызов функции
автоматически выбирает оптимальный алгоритм.
