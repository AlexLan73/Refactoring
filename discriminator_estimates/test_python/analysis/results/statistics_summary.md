# Статистика точности дискриминаторов по зонам

## Что здесь измеряется?

### Часть 1: sinc(x) — исходные данные (3-точечные методы)

Сдвигаем пик sinc(x) = sin(x)/x от x0 = −1.5 до +1.5 (301 точка).
На каждом шаге "снимаем" 3 отсчёта на сетке {−1, 0, +1} и пытаемся
**восстановить** положение пика по этим 3 точкам.

**5 методов**:
- **CG** — центр тяжести (2 точки, простой)
- **SD** — суммарно-разностный (2 точки, с коэффициентом c)
- **QA** — квадратичная аппроксимация (3 точки, парабола)
- **EA** — экспоненциальная аппроксимация (3 точки, Гауссиан)
- **AUTO** — автоматический: EA в normal зоне + E2 экстраполяция в extreme

**Метрики**: MAE = средняя ошибка, MaxErr = наихудшая, RMSE = среднеквадратичная,
Bias = систематическое смещение, Std = разброс, P95 = "в 95% случаев не хуже"

**Зоны**: normal (|x0|≤0.5), boundary (0.5<|x0|≤1.0), extreme (|x0|>1.0)

## Краткое резюме

### sinc(x) — 3-точечные (исходные данные)

```
Normal:     AUTO = EA (0.005) > QA (0.006) >> SD (0.205) > CG (0.226)
Extreme:    AUTO = E2 (0.256) > QA (0.585) ≈ CG (0.688)
AUTO — лучший из 3-точечных во ВСЕХ зонах!
```

### Hanning kernel — 5-точечные МНК (НОВОЕ) 🆕

```
Любой SNR:  5QA/5EA лучше EA3/QA3 в 1.5–2.0 раза!

  SNR < 25 дБ → 5QA (МНК-парабола)
  SNR > 25 дБ → 5EA (МНК-Гауссиан)
  
  5-точечные побеждают потому что Hanning kernel гладкий
  и парабола хорошо его приближает → малый bias + сглаживание шума.
```

### Финальная рекомендация

| Задача | Метод | Почему |
|--------|-------|--------|
| **Range FFT + Hanning** | **discr5ea** + zp×2 | Лучший при любом SNR |
| 3-точечный режим | **AUTO** (discr3_auto) | Автовыбор EA/E2/QA |
| Beam FFT (будущее) | Гибрид EA3+5QA | Голая ДН, парабола плохо ложится |
