Discriminator Estimates¶

Дискриминаторные оценки координат — поиск пика в зашумлённом сигнале

Полная документация модуля для определения частоты/координаты пика в FFT-спектре или ДН антенны с суб-бинной точностью.

Модуль:

discriminator_estimates

Организация:

Версия:

2.0 (2026-04-06)

О чём этот модуль¶

Библиотека отвечает на один очень практичный вопрос:

«У меня есть несколько точек измерения сигнала. Где находится истинный максимум — между точками?»

Под «точками» понимаются:

  • отсчёты диаграммы направленности (ДН) антенны для оценки угловых координат цели в РЛС;

  • бины спектра FFT для оценки точной частоты одиночного гармонического сигнала в зашумлённой записи.

В обоих случаях математика одинаковая, и здесь собраны 6 проверенных методов — от классических 2-точечных до новых 5-точечных МНК-аппроксимаций.

Аналогия из жизни

Представь, что ты стоишь на стадионе в шумной толпе и пытаешься услышать, на какой ноте поёт один конкретный певец. Толпа — это шум. Голос певца — наш полезный сигнал. У тебя есть 3–5 микрофонов, расставленных по частотной шкале. Задача: по этим 3–5 числам угадать, где точно находится голос певца.

Главная рекомендация¶

Совет

Используй discr5ea (МНК-Гауссиан по 5 точкам) с fallback на discr5qa и обязательным zero-padding ×2 + окном Hanning.

Это даёт точность ~0.6 Гц при SNR = 20 дБ — в 20 раз лучше, чем без интерполяции, и в 2 раза лучше классической 3-точечной discr3ea.

Подробное обоснование — на странице Рекомендации.

Быстрый старт¶

Суб-бинная оценка частоты по 5 точкам спектра (рекомендация):

#include "discr5ea.h"
#include <complex.h>
#include <stdlib.h>

// X[]      — комплексный спектр после FFT (длина N_padded)
// k_max    — индекс максимума |X(k)| (грубая оценка)
// fs       — частота дискретизации [Гц]
// N_padded — длина FFT с zero-padding ×2

double y[5], xs[5];
for (int i = 0; i < 5; ++i) {
  y[i]  = cabs(X[k_max - 2 + i]);   // |X(k-2)| .. |X(k+2)|
  xs[i] = (double)(i - 2);          // {-2, -1, 0, 1, 2}
}

double delta;   // суб-бинное смещение в единицах padded-шага
int ok = discr5ea(y, xs, &delta);   // МНК-Гауссиан по 5 точкам
if (ok != EXIT_SUCCESS) {
  // fallback: какая-то амплитуда ≤ 0 → 5QA по амплитудам
  discr5qa(y, xs, &delta);
}

double f_hz = (k_max + delta) * fs / (double)N_padded;

Классический 3-точечный QA на угловой ДН:

#include "discrqa.h"

double A1 = 0.75, A2 = 2.75, A3 = 2.75;
double x1 = -1.0, x2 = 0.0,  x3 = 1.0;
double xe = discr3qa(A1, A2, A3, x1, x2, x3);
// xe ≈ 0.5 — оценка положения пика между точками
