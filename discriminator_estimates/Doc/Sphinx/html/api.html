API Reference

signal_gen

Генерация тестового сигнала и вычисление оконного FFT.

tsignal.signal_gen.get_window(name: str, N: int) → ndarray[исходный код]

Вернуть оконную функцию по имени.

Встроенные (numpy): "hamming", "hann", "bartlett", "blackman" Scipy (если установлен): "kaiser8", "kaiser14", "flattop", "nuttall"

Параметры:

  • name (str) – Название окна (регистронезависимо)

  • N (int) – Длина окна

Тип результата:

np.ndarray, shape (N,)

tsignal.signal_gen.create_signal(fsin: float, fd: float, N: int, a: float = 1.0, an: float = 1e-09) → ndarray[исходный код]

Создать комплексный экспоненциальный сигнал с шумом.

Генерирует N отсчётов комплексной огибающей:

s(t) = a · exp(j·2π·fsin·t) + noise

Параметры:

  • fsin (float) – Частота сигнала, Гц

  • fd (float) – Частота дискретизации, Гц

  • N (int) – Количество отсчётов

  • a (float) – Амплитуда сигнала (по умолчанию 1.0)

  • an (float) – СКО шума (по умолчанию 1e-9 — практически без шума)

Результат:

Комплексная огибающая сигнала, N отсчётов

Тип результата:

np.ndarray (complex128)

tsignal.signal_gen.compute_fft(signal: ndarray, nfft: int, window: str = 'hamming') → ndarray[исходный код]

Вычислить оконный FFT спектр сигнала.

FFT = fftshift( fft(signal · win(N), nfft) )

Параметры:

  • signal (np.ndarray) – Комплексный входной сигнал (N отсчётов)

  • nfft (int) – Размер FFT

  • window (str) – Имя окна (см. get_window()). По умолчанию "hamming".

Результат:

Комплексный спектр (fftshift), nfft точек

Тип результата:

np.ndarray (complex128)

tsignal.signal_gen.make_freq_axis(nfft: int, fd: float) → ndarray[исходный код]

Построить частотную ось FFT после fftshift.

Параметры:

  • nfft (int) – Размер FFT

  • fd (float) – Частота дискретизации, Гц

Результат:

Частоты, Гц, nfft точек: от -fd/2 до fd/2 - df

Тип результата:

np.ndarray

discriminators

Частотные дискриминаторы FFT-спектра.

Точная реализация по C-исходникам из проекта discriminator_estimates (Refactoring/discriminator_estimates/src/).

Методы:

lay — 3-точечный Jacobsen: -Re{(Sp-Sm)/(2S0-Sm-Sp)} * fd/N (fcalcdelay.m) exp — 3-точечный, парабола на log|S| с проверками (discr3ea.c) sqr — 3-точечный, парабола по формуле Ao (discr3qa.c) sd — 2-точечный, суммарно-разностный (discrsd.c) cg — 2-точечный, центр тяжести (discrcg.c)

tsignal.discriminators.discriminator_lay(fft_data: ndarray, f_axis: ndarray) → tuple[float, float][исходный код]

Метод «Лайонса» — формула Jacobsen из fcalcdelay.m.

sigmapeak = (ffts(k+1)-ffts(k-1)) / (2*ffts(k)-ffts(k-1)-ffts(k+1)) f = fmax - real(sigmapeak) * fd / count

tsignal.discriminators.discriminator_exp(fft_data: ndarray, f_axis: ndarray) → tuple[float, float][исходный код]

Экспоненциальная 3-точечная интерполяция — из discr3ea.c.

Алгоритм:

  1. Сортировка 3 точек по x (координате)

  2. Проверка на выпуклость (максимум в центре)

  3. Логарифмирование: z = ln(A)

  4. Парабола z(x), вершина: xe = 0.5 * a / b

  5. Ограничение на вылет за пределы ±1.5·(f3-f1) от краёв

tsignal.discriminators.discriminator_sqr(fft_data: ndarray, f_axis: ndarray) → tuple[float, float][исходный код]

Квадратичная 3-точечная интерполяция — из discr3qa.c.

Формула (через отношение Ao):

Ao = (A2 - A1) / (A2 - A3) xe = 0.5 * ((Ao-1)*x2² - Ao*x3² + x1²) / ((Ao-1)*x2 - Ao*x3 + x1)

Граничные случаи обрабатываются как в оригинале.

tsignal.discriminators.discriminator_sd(fft_data: ndarray, f_axis: ndarray) → tuple[float, float][исходный код]

2-точечный суммарно-разностный — из discrsd.c.

Из fcalcdelay.m:

f = discr2sd(0.132497, P_max^2, P_second^2, f_max, f_second)

Из discrsd.c:

xc = (x1 + x2) / 2 dx = c * (A2 - A1) / (A2 + A1) xe = xc + dx

tsignal.discriminators.discriminator_cg(fft_data: ndarray, f_axis: ndarray) → tuple[float, float][исходный код]

2-точечный центр тяжести — из discrcg.c.

Из fcalcdelay.m:

f = discr2cg(a_max, a_second, f_max, f_second)

Из discrcg.c:

xe = (A1*x1 + A2*x2) / (A1 + A2)
