C API Reference¶

Справочник публичного C API модуля discriminator_estimates

1. discrcg.h — Центр тяжести¶

double discr2cg(double A1, double A2, double x1, double x2)¶

Двухточечный дискриминатор, центр тяжести.

Формула: \(x_e = \dfrac{A_1 x_1 + A_2 x_2}{A_1 + A_2}\).

Параметры:

  • A1 – амплитуда первой точки

  • A2 – амплитуда второй точки

  • x1 – координата первой точки

  • x2 – координата второй точки

Результат:

оценка координаты. При \(A_1 + A_2 = 0\) возвращает \((x_1 + x_2)/2\).

double discr3cg(double A1, double A2, double A3, double x1, double x2, double x3)¶

Трёхточечный дискриминатор, центр тяжести.

Формула: \(x_e = \dfrac{A_1 x_1 + A_2 x_2 + A_3 x_3}{A_1 + A_2 + A_3}\).

Результат:

оценка координаты. При сумме = 0 возвращает \(x_2\).

double discrncg(const double *A, const double *x, unsigned int N)¶

N-точечный дискриминатор, центр тяжести.

Параметры:

  • A – массив амплитуд (N элементов)

  • x – массив координат (N элементов)

  • N – число точек

Результат:

оценка координаты. При sum(A) = 0 возвращает x[N/2].

Пример:

#include "discrcg.h"

double A[3] = {0.5, 2.0, 1.5};
double x[3] = {-1.0, 0.0, 1.0};
double xe = discrncg(A, x, 3);
// xe ≈ 0.25

2. discrsd.h — Суммарно-разностный¶

double discr2sd(double c, double A1, double A2, double x1, double x2)¶

Двухточечный суммарно-разностный дискриминатор.

Формула: \(x_e = \dfrac{x_1 + x_2}{2} + c \cdot \dfrac{A_2 - A_1}{A_2 + A_1}\).

Параметры:

  • c – коэффициент крутизны дискриминаторной характеристики

  • A1 – амплитуда первой точки

  • A2 – амплитуда второй точки

  • x1 – координата первой точки

  • x2 – координата второй точки

Результат:

оценка координаты. При \(A_1 + A_2 = 0\) возвращает \((x_1 + x_2)/2\).

Пример:

#include "discrsd.h"

double c = 0.132497;  // подобрано для конкретной ДН
double xe = discr2sd(c, 0.8, 1.2, -0.5, 0.5);

3. discrqa.h — Парабола по 3 точкам¶

double discr3qa(double A1, double A2, double A3, double x1, double x2, double x3)¶

Трёхточечный дискриминатор, парабола по 3 точкам.

Через 3 точки проводится парабола \(y = ax^2 + bx + c\), находится её вершина.

Вспомогательная формула: \(A_o = \dfrac{A_2 - A_1}{A_2 - A_3}\).

Граничные случаи обрабатываются через DBL_EPSILON (см. Формулы).

Результат:

оценка координаты \(x_e\).

Пример:

#include "discrqa.h"

double A1 = 0.75, A2 = 2.75, A3 = 2.75;
double x1 = -1.0, x2 = 0.0,  x3 = 1.0;
double xe = discr3qa(A1, A2, A3, x1, x2, x3);
// xe ≈ 0.5 — оценка положения пика между точками

4. discrea.h — Гауссиан по 3 точкам (EA)¶

int discr3ea(double A1, double A2, double A3, double x1, double x2, double x3, double *xe)¶

Трёхточечный дискриминатор, экспоненциальная (гауссова) аппроксимация.

Аппроксимирует отсчёты ДН функцией \(y = A_{\max} \exp(-a(x-x_0)^2)\):

  1. Логарифмирует амплитуды: \(z_i = \ln A_i\).

  2. Аппроксимирует параболой \(z(x) = \alpha + \beta x + \gamma x^2\).

  3. Находит вершину \(x_e = \alpha / (2\beta)\).

Параметры:

  • A1-A3 – амплитуды (должны быть > 0)

  • x1-x3 – координаты

  • xe – [out] оценка координаты

Результат:

EXIT_SUCCESS (0) при успехе, EXIT_FAILURE (1) при ошибке.

double discr3eaY(double A1, double A2, double x1, double x2, double xe)¶

Уточнение амплитуды по найденной координате \(x_e\).

Результат:

уточнённое значение амплитуды \(y_e\) в точке \(x_e\). При \(A_1 < \varepsilon\) или \(A_2 < \varepsilon\) возвращает 0.0.

Пример:

#include "discrea.h"
#include <stdlib.h>

double A1 = 0.25, A2 = 1.0, A3 = 0.25;
double xe;
int ok = discr3ea(A1, A2, A3, -1.0, 0.0, 1.0, &xe);
if (ok == EXIT_SUCCESS) {
  double ye = discr3eaY(A2, A3, 0.0, 1.0, xe);
}

5. discr5ea.h — МНК по 5 точкам ★¶

int discr5ea(const double A[5], const double x[5], double *xe)¶

Рекомендуемый по умолчанию. Пятиточечный МНК-гауссиан.

Аппроксимирует 5 равноотстоящих отсчётов гауссианом методом наименьших квадратов. Логарифмирует амплитуды и применяет замкнутую МНК-формулу по параболе в log-пространстве.

Параметры:

  • A – массив из 5 амплитуд (все > 0), центр = A[2]

  • x – массив из 5 координат, равноотстоящих

  • xe – [out] оценка координаты \(x_e\)

Результат:

EXIT_SUCCESS (0) при успехе, EXIT_FAILURE (1) при ошибке.

int discr5qa(const double A[5], const double x[5], double *xe)¶

Пятиточечный МНК-параболический дискриминатор.

Аналогичен discr5ea, но работает напрямую с амплитудами (без логарифмирования). Используется как fallback, когда \(A_i \leq 0\).

Параметры:

  • A – массив из 5 амплитуд (могут быть любые)

  • x – массив из 5 координат, равноотстоящих

  • xe – [out] оценка координаты

Результат:

EXIT_SUCCESS / EXIT_FAILURE.

Пример — правильное использование с fallback:

#include "discr5ea.h"
#include <complex.h>
#include <stdlib.h>

double y[5], xs[5];
for (int i = 0; i < 5; ++i) {
  y[i]  = cabs(X[k_max - 2 + i]);
  xs[i] = (double)(i - 2);
}

double delta;
int ok = discr5ea(y, xs, &delta);
if (ok != EXIT_SUCCESS) {
  discr5qa(y, xs, &delta);      // fallback
}

double f_hz = (k_max + delta) * fs / (double)N_padded;

6. discr_auto.h — Автовыбор метода¶

int discr_is_monotonic(double A1, double A2, double A3)¶

Проверка монотонности трёх отсчётов.

Результат:

1 — монотонные (пик за сеткой), 0 — есть перегиб (пик внутри).

int discr3_extrap_gauss(double A1, double A2, double A3, double x1, double x2, double x3, double *xe)¶

Гауссова экстраполяция (E2) для монотонных случаев.

Алгоритм как discr3ea, но без проверки выпуклости и с расширенным диапазоном допустимого вылета (2× вместо 0.5×).

Результат:

0 — успех, 1 — ошибка.

int discr3_auto(double A1, double A2, double A3, double x1, double x2, double x3, double *xe)¶

Автоматический дискриминатор с переключением.

Алгоритм выбора:

if is_monotonic(A1, A2, A3):
    xe = extrap_gauss(...)        // E2
else:
    ret = discr3ea(...)           // EA (лучший)
    if ret == FAILURE:
        xe = discr3qa(...)        // QA fallback
Результат:

  • 0 — EA успех

  • 1 — использован QA fallback

  • 2 — использована экстраполяция E2

  • 3 — экстраполяция E2 не сошлась (xe = x2)

Пример:

#include "discr_auto.h"

double xe;
int how = discr3_auto(A1, A2, A3, x1, x2, x3, &xe);
switch (how) {
  case 0: printf("EA normal\n"); break;
  case 1: printf("QA fallback\n"); break;
  case 2: printf("E2 extrapolation\n"); break;
  case 3: printf("E2 failed, xe=x2\n"); break;
}

7. discr_common.h — Обёртки discr2 / discr3¶

Высокоуровневые обёртки для удобного вызова 2pt / 3pt дискриминаторов по 2D-массивам (для РЛС-моноимпульсного канала).

Используют DISCRTYPE enum из discr_types.h:

typedef enum {
  DT_CG = 0,   // центр тяжести
  DT_SD = 1,   // сумма-разность
  DT_QA = 2,   // парабола
  DT_EA = 3,   // гауссиан
} DISCRTYPE;

ERR-003 fix

После рефакторинга обёртки не модифицируют входные массивы — координаты умножаются на коэффициент \(c = 2\pi dx / \lambda\) при передаче в дискриминатор.

Значение

Смысл

Python API (будущее)¶

Примечание

Python-биндинги в разработке. Планируется два подхода:

  • ctypes — быстрый старт, загрузка .so/.dll напрямую

  • pybind11 — когда C++ обёртка стабилизируется

Прототип реализован в test_python/test_discriminators.py, см. Тесты.
