Трехточечный дискриминатор, аппроксимация квадратичной зависимостью (Quadratic).

Аппроксимирует 3 точки параболой y = a*x² + b*x + c и находит вершину.

Формулы: Ao = (A2 - A1) / (A2 - A3) xe = 0.5 * ((Ao-1)*x2² - Ao*x3² + x1²) / ((Ao-1)*x2 - Ao*x3 + x1)

Граничные случаи:

  • A2 = A3: если A1 = A2 → x2; если A1 > A2 → x1; иначе → (x2+x3)/2
  • A1 = A2: если A3 > A2 → x3; иначе → (x1+x2)/2
