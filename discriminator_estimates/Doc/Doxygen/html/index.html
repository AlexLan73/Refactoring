Назначение

Библиотека реализует дискриминаторные оценки координат цели по отсчётам диаграммы направленности (ДН) антенны. Применяется в комплексе программ первичной обработки (КППО) для вычисления текущих замеров координат (ТЗК).

Физический смысл

Диаграмма направленности антенны описывается функцией, близкой к:

\[\text{sinc}(x) = \frac{\sin(x)}{x} \]

Антенна "смотрит" в несколько направлений (2-3 луча) и измеряет амплитуду принятого сигнала в каждом. По этим отсчётам дискриминатор оценивает истинное положение цели между лучами.

C (3-точечный QA дискриминатор):

#include "discrqa.h"

double xe = discr3qa(A1, A2, A3, x1, x2, x3);
// xe ≈ 0.5 — оценка координаты цели

Зависимость ошибки от шага сетки

Рис. 3: Ошибка оценки при разных шагах сетки

При мелкой сетке (шаг < 0.5) все методы дают хорошую точность. При крупной сетке (шаг > 1.5) экспоненциальный метод (EA) значительно точнее квадратичного (QA) и центра масс (CG).
