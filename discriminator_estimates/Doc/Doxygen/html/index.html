Назначение

Библиотека реализует дискриминаторные оценки координат цели по отсчётам диаграммы направленности (ДН) антенны. Применяется в КППО для ТЗК.

Физический смысл

Диаграмма направленности антенны описывается функцией, близкой к:

\[ \text{sinc}(x) = \frac{\sin(x)}{x} \]

Антенна "смотрит" в несколько направлений (2-3 луча) и измеряет амплитуду. По отсчётам дискриминатор оценивает истинное положение цели между лучами.
