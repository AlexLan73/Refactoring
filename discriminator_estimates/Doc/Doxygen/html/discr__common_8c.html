Выделение подматрицы 2x2 из 3x3 для перехода на 2-точечный дискриминатор.

Выбирает квадрант 2x2 с максимальной суммарной амплитудой. Используется когда 3-точечный дискриминатор невозможен (checkuse3d вернул 0, вогнутость и т.д.).

Аргументы
maz3[in] азимутальные координаты 3x3
mel3[in] угломестные координаты 3x3
mval3[in] амплитуды 3x3
maz2[out] азимутальные координаты 2x2
mel2[out] угломестные координаты 2x2
mval2[out] амплитуды 2x2
Возвращает
0 — успех

Трехточечный дискриминатор (обёртка).

Принимает матрицы ndnx x ndny координат и амплитуд ДН. Поддерживает типы: DT_CG, DT_QA, DT_EA.
