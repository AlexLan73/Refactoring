Двухточечный дискриминатор, центр тяжести.

Формула: \( x_e = \frac{A_1 \cdot x_1 + A_2 \cdot x_2}{A_1 + A_2} \)

Аргументы
A1амплитуда первой точки
A2амплитуда второй точки
x1координата первой точки
x2координата второй точки
Возвращает
оценка координаты; при A1+A2=0 возвращает (x1+x2)/2

Формула: xe = (A1*x1 + A2*x2) / (A1 + A2)

Аргументы
A1амплитуда первой точки
A2амплитуда второй точки
x1координата первой точки
x2координата второй точки
Возвращает
оценка координаты; при A1+A2=0 возвращает середину (x1+x2)/2

Дискриминатор для произвольного числа точек, центр тяжести.

Формула: \( x_e = \frac{\sum_{i=0}^{N-1} A_i \cdot x_i}{\sum_{i=0}^{N-1} A_i} \)

Аргументы
Aмассив амплитуд (N элементов)
xмассив координат (N элементов)
Nчисло точек
Возвращает
оценка координаты; при sum(A)=0 возвращает x[N/2]

Формула: xe = sum(A[i]*x[i]) / sum(A[i])

Аргументы
Aмассив амплитуд точек (N элементов)
xмассив координат точек (N элементов)
Nчисло точек
Возвращает
оценка координаты; при sum(A)=0 возвращает x[N/2]
