Реализует два метода МНК по 5 равноотстоящим точкам:

  • discr5ea: гауссов (в лог-масштабе)
  • discr5qa: параболический (в линейном масштабе)

Математика: Нормируем координаты xi -> {-2, -1, 0, 1, 2}. Для переопределённой системы y = a*x^2 + b*x + c (5 уравнений, 3 неизвестных) нормальные уравнения (A^T * A) * beta = A^T * y дают замкнутые формулы:

a = (2*y1 - y2 - 2*y3 - y4 + 2*y5) / 14 b = (-2*y1 - y2 + y4 + 2*y5) / 10 xe = -b / (2*a) (в нормированных единицах)

Затем обратная нормировка: xe_real = x3 + h * xe_norm

Пятиточечный дискриминатор, МНК-гауссова аппроксимация (LSQ Gaussian 5pt).

Аппроксимирует 5 равноотстоящих отсчётов гауссианом методом наименьших квадратов:

\[ y = A_{max} \exp\!\bigl(-a\,(x - x_0)^2\bigr) \]

Алгоритм:

  1. Логарифмирование: \( z_i = \ln A_i \)
  2. МНК-параболическая аппроксимация \( z(x) = a\,x^2 + b\,x + c \) по 5 равноотстоящим точкам (переопределённая система 5 уравнений, 3 неизвестных)
  3. Вершина параболы: \( x_e = -b / (2a) \)

Формулы МНК для нормированных координат \( \{-2, -1, 0, 1, 2\} \):

\[ a = \frac{2z_1 - z_2 - 2z_3 - z_4 + 2z_5}{14} \]

\[ b = \frac{-2z_1 - z_2 + z_4 + 2z_5}{10} \]

\[ x_e = x_3 + h \cdot \frac{-b}{2a} \]

Преимущества перед discr3ea:

  • Устойчивость к шуму за счёт сглаживания (5 точек → 3 параметра)
  • Гауссова модель хорошо приближает sinc(x) вблизи пика
Предупреждения
Все амплитуды должны быть строго положительны ( \( A_i > 0 \)).
Точки должны быть равноотстоящими (постоянный шаг h).
Аргументы
Aмассив из 5 амплитуд [A1, A2, A3, A4, A5], центральная = A[2]
xмассив из 5 координат [x1, x2, x3, x4, x5], равноотстоящие
xe[out] оценка координаты \( x_e \)
Возвращает
EXIT_SUCCESS (0) при успехе, EXIT_FAILURE (1) при ошибке (при ошибке *xe содержит приближённое значение)
