Двухточечный дискриминатор, суммарно-разностный (Sum-Difference).

Формула:

\[ x_e = \frac{x_1 + x_2}{2} + c \cdot \frac{A_2 - A_1}{A_2 + A_1} \]

Аргументы
cкоэффициент суммарно-разностного дискриминатора
A1амплитуда первой точки
A2амплитуда второй точки
x1координата первой точки
x2координата второй точки
Возвращает
оценка координаты \( x_e \); при \( A_1+A_2=0 \) возвращает середину \( (x_1+x_2)/2 \)

Двухточечный дискриминатор, суммарно-разностный (Sum-Difference).

Формулы: xc = (x1 + x2) / 2 dx = c * (A2 - A1) / (A2 + A1) xe = xc + dx

Аргументы
cкоэффициент суммарно-разностного дискриминатора
A1амплитуда первой точки
A2амплитуда второй точки
x1координата первой точки
x2координата второй точки
Возвращает
оценка координаты; при A1+A2=0 возвращает середину (x1+x2)/2
