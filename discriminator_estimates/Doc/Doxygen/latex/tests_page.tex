\begin{DoxyItemize}
\item Голубая линия — sinc(x -\/ 0.\+25) (ДН антенны, пик смещён на 0.\+25)
\item Жёлтые точки — 3 отсчёта на сетке \{-\/1, 0, +1\}
\item Вертикальные линии — оценки дискриминаторов
\item Зелёная пунктирная — истинное положение пика
\end{DoxyItemize}



График показывает зависимость абсолютной ошибки от смещения пика\+:
\begin{DoxyItemize}
\item CG (красный) и SD (фиолетовый) — линейный рост ошибки
\item QA (бирюзовый) — ошибка \texorpdfstring{$<$}{<} 0.\+02 при любом смещении
\item EA (жёлтый) — самая маленькая ошибка
\end{DoxyItemize}



Влияние шага сетки на точность\+:
\begin{DoxyItemize}
\item При мелкой сетке (\texorpdfstring{$<$}{<} 0.\+5) все методы точны
\item При крупной сетке (\texorpdfstring{$>$}{>} 1.\+5) EA значительно лучше остальных
\end{DoxyItemize}
