Дискриминаторы модуля применяются не только для оценки угловых координат по ДН антенны, но и для {\bfseries{точной оценки частоты}} по FFT-\/спектру комплексного сигнала.

\begin{quote}
{\bfseries{Рекомендация для современного pipeline\+:}} для CPU FFT-\/задачи наилучшим выбором является \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}{\ttfamily discr5ea} (МНК-\/Гауссиан по 5 точкам)\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*} с fallback на {\ttfamily discr5qa} и обязательным {\ttfamily zero-\/padding ×2} + окном Hanning. Это даёт ошибку \texorpdfstring{$\sim$}{\string~}0.6 Гц при SNR = 20 дБ — в 20 раз лучше argmax и в 2 раза лучше классической 3-\/точечной EA3. Полное обоснование, дерево решений и Quick reference — на странице \doxylink{recommendations_page}{Итоговые рекомендации}. \end{quote}

\hypertarget{fft_page_autotoc_md3}{}\doxysubsection{\texorpdfstring{5 методов оценки частоты}{5 методов оценки частоты}}\label{fft_page_autotoc_md3}
\hypertarget{fft_page_autotoc_md4}{}\doxysubsubsection{\texorpdfstring{3-\/точечные (бины k-\/1, k, k+1)}{3-точечные (бины k-1, k, k+1)}}\label{fft_page_autotoc_md4}
{\bfseries{EXP}} (discr3ea.\+c — парабола на log\texorpdfstring{$\vert$}{|}\+S\texorpdfstring{$\vert$}{|})\+:  \[
z_i = \ln|S_i|, \quad
\alpha = z_1(f_2^2-f_3^2) + z_2(f_3^2-f_1^2) + z_3(f_1^2-f_2^2)
\]  \[
\beta = z_1(f_2-f_3) + z_2(f_3-f_1) + z_3(f_1-f_2), \quad
f_{est} = \frac{\alpha}{2\beta}
\]

{\bfseries{SQR}} (discr3qa.\+c — парабола на \texorpdfstring{$\vert$}{|}\+S\texorpdfstring{$\vert$}{|})\+:  \[
A_o = \frac{A_2 - A_1}{A_2 - A_3}, \quad
f_{est} = \frac{(A_o-1)f_2^2 - A_o f_3^2 + f_1^2}{2\bigl((A_o-1)f_2 - A_o f_3 + f_1\bigr)}
\]

{\bfseries{LAY}} (Jacobsen, fcalcdelay.\+m)\+:  \[
\sigma = \frac{S_{k+1} - S_{k-1}}{2 S_k - S_{k-1} - S_{k+1}}, \quad
f_{est} = f_k - \operatorname{Re}\{\sigma\} \cdot \frac{f_d}{N}
\]\hypertarget{fft_page_autotoc_md5}{}\doxysubsubsection{\texorpdfstring{2-\/точечные (top-\/2 bins по амплитуде)}{2-точечные (top-2 bins по амплитуде)}}\label{fft_page_autotoc_md5}
{\bfseries{SD}} (\doxylink{discrsd_8c}{discrsd.\+c})\+:  \[
f_{est} = \frac{f_1+f_2}{2} + c \cdot \frac{P_2 - P_1}{P_2 + P_1}, \quad c = 0.132497
\]

{\bfseries{CG}} (\doxylink{discrcg_8c}{discrcg.\+c})\+:  \[
f_{est} = \frac{A_1 f_1 + A_2 f_2}{A_1 + A_2}
\]
