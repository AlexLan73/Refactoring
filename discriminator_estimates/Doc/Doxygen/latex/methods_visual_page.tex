\hypertarget{methods_visual_page_autotoc_md74}{}\doxysubsubsection{\texorpdfstring{Sweep по всей зоне — все методы вместе}{Sweep по всей зоне — все методы вместе}}\label{methods_visual_page_autotoc_md74}


{\bfseries{Что происходит на кадре\+:}}
\begin{DoxyItemize}
\item по горизонтали — координата;
\item голубая кривая — sinc/\+Hanning kernel с движущимся пиком;
\item зелёный пунктир — {\bfseries{истинное}} положение пика $ x_0 $;
\item цветные вертикальные линии — оценки разных методов;
\item крайние точки слева/справа — отсчёты, которые «видит» алгоритм.
\end{DoxyItemize}

{\bfseries{На что смотреть\+:}}
\begin{DoxyItemize}
\item В {\bfseries{центральной зоне}} ( $ |x_0| \le 0.5 $ шага) — все методы вместе.
\item Ближе к краям — 3-\/точечные методы (CG, SD, QA, EA) {\bfseries{разбегаются}}\+: пик уходит из их «зоны видимости».
\item 5-\/точечные (5QA, 5EA) держат пик намного дольше и не отрываются от зелёной пунктирной линии.
\end{DoxyItemize}

\hypertarget{methods_visual_page_autotoc_md76}{}\doxysubsubsection{\texorpdfstring{Крупный план — нормальная зона}{Крупный план — нормальная зона}}\label{methods_visual_page_autotoc_md76}


То же самое, но {\bfseries{крупным планом}}, в пределах нормальной зоны $ |x_0| \le 1 $ шаг. Здесь видно даже мелкие различия между методами\+:
\begin{DoxyItemize}
\item EA сидит почти на самой зелёной линии,
\item QA немного дрожит относительно неё,
\item CG/\+SD ощутимо смещены.
\end{DoxyItemize}

\DoxyHorRuler{0}
\hypertarget{methods_visual_page_autotoc_md78}{}\doxysubsubsection{\texorpdfstring{Нарастание шума}{Нарастание шума}}\label{methods_visual_page_autotoc_md78}


{\bfseries{Что происходит\+:}} пик стоит на месте, а шум плавно нарастает (SNR с 40 дБ падает до 0 дБ). Каждый кадр — новая случайная реализация шума.

{\bfseries{На что смотреть\+:}}
\begin{DoxyItemize}
\item При {\bfseries{высоком SNR}} все оценки сидят рядом с истинным пиком.
\item При {\bfseries{низком SNR}} оценки начинают «дрожать». Чем стабильнее метод — тем меньше амплитуда дрожания.
\item 5EA / 5QA дрожат заметно меньше, чем 3-\/точечные собратья — это визуальное подтверждение их шумоустойчивости из \doxylink{recommendations_page}{Итоговые рекомендации}.
\end{DoxyItemize}

\DoxyHorRuler{0}
\hypertarget{methods_visual_page_autotoc_md80}{}\doxysubsubsection{\texorpdfstring{Резкое изменение пика}{Резкое изменение пика}}\label{methods_visual_page_autotoc_md80}


Тестируем переходный процесс\+: пик {\bfseries{мгновенно}} скачет с одной координаты на другую. Никакого сглаживания между кадрами — каждый кадр это новый независимый замер.

{\bfseries{Что показывает\+:}} все методы {\bfseries{немедленно}} реагируют (нет инерции — дискриминаторы не имеют памяти). Полезно как контроль того, что в реализации нет случайной фильтрации.

\hypertarget{methods_visual_page_autotoc_md84}{}\doxysubsection{\texorpdfstring{Раздел 2. Карты точности — 3\+D и heatmap}{Раздел 2. Карты точности — 3D и heatmap}}\label{methods_visual_page_autotoc_md84}
\hypertarget{methods_visual_page_autotoc_md85}{}\doxysubsubsection{\texorpdfstring{5\+EA\+: точность как функция (x0, SNR)}{5EA: точность как функция (x0, SNR)}}\label{methods_visual_page_autotoc_md85}
Это {\bfseries{самая важная карта}} — показывает, как ведёт себя рекомендуемый метод 5EA в {\bfseries{двумерном пространстве}} ключевых параметров\+: смещение пика $ x_0 $ и уровень шума SNR.





{\bfseries{Что видно\+:}}


\begin{DoxyItemize}
\item {\bfseries{Левая нижняя зона}} (малый $ x_0 $, высокий SNR) — тёмная = маленькая ошибка. Это идеальные условия\+: пик по центру окна, шум слабый.
\item {\bfseries{Правая зона}} (большой $ x_0 $) — резкий рост ошибки. Это {\bfseries{монотонная зона}} (пик вышел за пределы 3-\/точечного окна).
\item {\bfseries{Верхняя зона}} (низкий SNR) — ошибка растёт при любом $ x_0 $\+: шум слишком сильный.
\item {\bfseries{Голубая линия}} на 3D-\/поверхности — граница монотонной зоны для 3pt методов. Правее этой линии 5EA продолжает работать, а QA(3pt) уже сыплется.
\end{DoxyItemize}

{\bfseries{Практический вывод\+:}} чтобы получить ошибку \texorpdfstring{$<$}{<} 0.\+05 шага, нужно одновременно\+:
\begin{DoxyItemize}
\item удерживать {\bfseries{SNR \texorpdfstring{$>$}{>} 15 дБ}} И
\item держать \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*} $ |x_0| < 1.5 $ шага\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*} от центра окна.
\end{DoxyItemize}

\hypertarget{methods_visual_page_autotoc_md87}{}\doxysubsubsection{\texorpdfstring{Сравнение 4 классических методов на одной картинке}{Сравнение 4 классических методов на одной картинке}}\label{methods_visual_page_autotoc_md87}


Четыре 3D-\/поверхности рядом. Сравниваем — у какого метода «долина» {\bfseries{глубже}} (меньше ошибка) и {\bfseries{шире}} (больше допустимый диапазон $ x_0 $).

{\bfseries{Типичный вывод\+:}}
\begin{DoxyItemize}
\item {\bfseries{EA}} — самая глубокая долина (точнее всех на чистом сигнале);
\item {\bfseries{CG}} — самая широкая, но мелкая (грубо, но устойчиво);
\item {\bfseries{QA}} — промежуточный вариант;
\item {\bfseries{SD}} — нестабилен при неправильном выборе коэффициента $ c $.
\end{DoxyItemize}

\hypertarget{methods_visual_page_autotoc_md96}{}\doxysubsubsection{\texorpdfstring{Ошибка vs смещение пика — sweep}{Ошибка vs смещение пика — sweep}}\label{methods_visual_page_autotoc_md96}


Базовый бенчмарк. По горизонтали — истинное смещение пика $ x_0 $, по вертикали — модуль ошибки. {\bfseries{Чем ниже кривая, тем точнее метод.}}

Наблюдения\+:
\begin{DoxyItemize}
\item {\bfseries{CG}} и {\bfseries{SD}} — почти линейно растут (классическая систематическая ошибка взвешенного среднего);
\item {\bfseries{QA}} — почти 0 в нормальной зоне, потом резкий рост;
\item {\bfseries{EA}} — самая низкая в нормальной зоне.
\end{DoxyItemize}
