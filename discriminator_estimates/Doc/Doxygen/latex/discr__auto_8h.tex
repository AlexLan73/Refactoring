\doxysubsection{Подробное описание}
Автоматический выбор дискриминатора с экстраполяцией. 

\begin{DoxyAuthor}{Автор}
Кодо (AI Assistant) 
\end{DoxyAuthor}
\begin{DoxyDate}{Дата}
2026-\/04-\/02 
\end{DoxyDate}
Решает проблему монотонных данных (пик за сеткой), когда стандартные дискриминаторы (EA, QA, CG) работают плохо (ошибка \texorpdfstring{$>$}{>} 25\% шага).

Стратегия\+:
\begin{DoxyEnumerate}
\item Проверить монотонность 3 отсчётов
\item Если монотонные → Гауссова экстраполяция (E2)
\item Если нормальные → EA (экспоненциальный дискриминатор)
\item Если EA вернул FAILURE → QA (квадратичный fallback)
\end{DoxyEnumerate}

Обоснование (из аналитического исследования)\+:
\begin{DoxyItemize}
\item EA\+: MAE = 0.\+005 в нормальной зоне (лучший метод)
\item E2\+: MAE = 0.\+148 для монотонных случаев (лучше E1 и E3)
\item QA\+: MAE = 0.\+006, fallback если EA не сходится
\item Без авто\+: 67\% случаев попадают в монотонную зону! 
\end{DoxyItemize}

Гауссова экстраполяция (E2) для монотонных случаев. 

Алгоритм — как discr3ea, но без проверки выпуклости и с расширенным диапазоном допустимого вылета (2× вместо 0.\+5× от размера сетки).

Логарифмирует амплитуды, аппроксимирует параболой в log-\/пространстве, находит вершину. Для монотонных данных вершина за пределами сетки — это ожидаемое и полезное поведение.

Из исследования\+: E2 MAE = 0.\+148 для монотонных случаев, что в 2.\+2 раза лучше E1 (парабола, MAE = 0.\+324) и в 3.\+4 раза лучше E3 (градиент, MAE = 0.\+500).

Проверка монотонности трёх отсчётов. 

Монотонные данные = нет локального максимума среди 3 точек. Это означает, что пик за пределами сетки.
