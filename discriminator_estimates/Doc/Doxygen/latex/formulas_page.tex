{\bfseries{Шаг 3}}\+: Проверка выпуклости
\begin{DoxyItemize}
\item Максимум должен быть в центре (а не на краю)
\item При вогнутости → {\ttfamily EXIT\+\_\+\+FAILURE}

{\bfseries{Шаг 4}}\+: Логарифмирование

\[
z_i = \ln(A_i), \quad i = 1, 2, 3
\]

Задача сводится к параболической аппроксимации в логарифмическом масштабе\+: $ z(x) = -a(x - x_0)^2 + \ln(A_{max}) $

{\bfseries{Шаг 5}}\+: Вершина параболы

\[
\alpha = z_1 (f_2^2 - f_3^2) + z_2 (f_3^2 - f_1^2) + z_3 (f_1^2 - f_2^2)
\]

\[
\beta = z_1 (f_2 - f_3) + z_2 (f_3 - f_1) + z_3 (f_1 - f_2)
\]

\[
x_e = \frac{\alpha}{2 \beta}
\]

где $ f_i $ — координаты после сортировки.

{\bfseries{Шаг 6}}\+: Ограничение на вылет

\[
f_1 - \frac{f_3 - f_1}{2} \leq x_e \leq f_3 + \frac{f_3 - f_1}{2}

Оценка не должна выходить за пределы области зондирования более чем на полшага.\hypertarget{formulas_page_autotoc_md30}{}\doxysubsubsection{\texorpdfstring{Уточнение амплитуды (discr3ea\+Y)}{Уточнение амплитуды (discr3eaY)}}\label{formulas_page_autotoc_md30}

\hypertarget{formulas_page_autotoc_md32}{}\doxysubsection{\texorpdfstring{5. МНК-\/парабола по 5 точкам — 5\+QA ($<$tt$>$discr5qa$<$/tt$>$)}{5. МНК-парабола по 5 точкам — 5QA (<tt>discr5qa</tt>)}}\label{formulas_page_autotoc_md32}
Идея\+: вместо параболы, идеально проходящей через {\bfseries{3}} точки, аппроксимируем параболой {\bfseries{5}} точек {\bfseries{по методу наименьших квадратов}} (МНК / LSQ). Получаем переопределённую систему — 5 уравнений на 3 неизвестных $ (a, b, c) $\+:

\[
A_i \approx a\,x_i^2 + b\,x_i + c, \qquad i = 1,\ldots,5
\]

В матричной форме\+:  \[
\mathbf{H}\,\boldsymbol{\theta} = \mathbf{y},
\quad
\mathbf{H} =
\begin{pmatrix}
x_1^2 & x_1 & 1 \\
x_2^2 & x_2 & 1 \\
x_3^2 & x_3 & 1 \\
x_4^2 & x_4 & 1 \\
x_5^2 & x_5 & 1
\end{pmatrix},
\quad
\boldsymbol{\theta} = \begin{pmatrix} a \\ b \\ c \end{pmatrix}.
\]

Решение МНК (нормальные уравнения)\+:  \[
(\mathbf{H}^{\!\top}\mathbf{H})\,\boldsymbol{\theta} = \mathbf{H}^{\!\top}\mathbf{y}.
\]\hypertarget{formulas_page_autotoc_md33}{}\doxysubsubsection{\texorpdfstring{Замкнутая формула для равноотстоящих \textbackslash{}f\$ x\+\_\+i = \textbackslash{}\{-\/2,-\/1,0,1,2\textbackslash{}\} \textbackslash{}f\$}{Замкнутая формула для равноотстоящих \textbackslash{}f\$ x\_i = \textbackslash{}\{-2,-1,0,1,2\textbackslash{}\} \textbackslash{}f\$}}\label{formulas_page_autotoc_md33}
Для равномерной сетки $ x_i = -2,-1,0,1,2 $ матрица $ \mathbf{H}^{\!\top}\mathbf{H} $ блочно-\/диагональна (так как сумма нечётных степеней равна нулю), и решение можно записать в {\bfseries{замкнутом виде}}\+:

\[
\boxed{
a = \frac{2A_1 - A_2 - 2A_3 - A_4 + 2A_5}{14}
}
\]  \[
\boxed{
b = \frac{-2A_1 - A_2 + A_4 + 2A_5}{10}
}
\]  \[
\boxed{
\delta = -\frac{b}{2a}, \qquad x_e = x_3 + h\,\delta
}
\]

где $ h $ — шаг сетки, $ x_3 $ — центральная точка.

{\bfseries{Свойства\+:}}
\begin{DoxyItemize}
\item Сложность\+: O(1) — 6 умножений + 7 сложений + 1 деление
\item Не требует никаких циклов и матричных вычислений
\item В нормальной зоне даёт ошибку \texorpdfstring{$\sim$}{\string~}2× меньше, чем 3-\/точечная QA
\end{DoxyItemize}



\begin{DoxyNote}{Заметки}
Защита\+: при $ |a| < 10^{-30} $ возвращается $ x_e = x_3 $ (данные плоские — оценить нельзя).
\end{DoxyNote}

\hypertarget{formulas_page_autotoc_md35}{}\doxysubsection{\texorpdfstring{6. МНК-\/Гауссиан по 5 точкам — 5\+EA ($<$tt$>$discr5ea$<$/tt$>$)}{6. МНК-Гауссиан по 5 точкам — 5EA (<tt>discr5ea</tt>)}}\label{formulas_page_autotoc_md35}

z_i = \ln A_i, \quad i = 1,\ldots,5

В лог-\/пространстве гауссиан превращается в {\bfseries{параболу}}\+:  \[
z(x) = -a\,(x - x_0)^2 + \ln A_{\max} = a\textnormal{\textquotesingle}x^2 + b\textnormal{\textquotesingle}x + c\textnormal{\textquotesingle}.
\]

{\bfseries{Шаг 2.}} К {\ttfamily z\mbox{[}\mbox{]}} применяем те же замкнутые МНК-\/формулы, что и в 5QA\+:  \[

b = \frac{-2z_1 - z_2 + z_4 + 2z_5}{10}
\]

{\bfseries{Шаг 3.}} Вершина параболы → оценка $ x_e $\+:  \[
\delta = -\frac{b}{2a}, \qquad x_e = x_3 + h\,\delta.

\hypertarget{formulas_page_autotoc_md44}{}\doxysubsection{\texorpdfstring{8. Обёртка discr3() — коэффициент пересчёта}{8. Обёртка discr3() — коэффициент пересчёта}}\label{formulas_page_autotoc_md44}
Для обёрток {\ttfamily \doxylink{discr__common_8h_ada0fa80ec1c0edd0e0135de4d4a01c5d}{discr3()}} и {\ttfamily \doxylink{discr__common_8h_ac4d4e1deafa0bcc5b3c0ba33b1190c43}{discr3\+\_\+()}} координаты пересчитываются\+:

\[
c = \frac{2 \pi \cdot dx}{\lambda}
\]

где $ dx $ — шаг антенной решётки, $ \lambda $ — длина волны.

Координаты масштабируются\+: $ x\textnormal{\textquotesingle} = x \cdot c $, оценка возвращается в исходных единицах\+: $ x_e = x\textnormal{\textquotesingle}_e / c $.

\begin{DoxyNote}{Заметки}
После рефакторинга (ERR-\/003 fix) входные массивы {\bfseries{не модифицируются}} — координаты умножаются на $ c $ при передаче в дискриминатор, а не в массиве. 
\end{DoxyNote}
