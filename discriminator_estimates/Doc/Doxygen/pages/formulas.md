# Математические формулы {#formulas_page}

## Полное описание формул дискриминаторных оценок

---

## 1. Центр тяжести (CG — Center of Gravity)

Простейший метод. Координата оценивается как взвешенное среднее.

### 2-точечный (discr2cg)

\f[
x_e = \frac{A_1 \cdot x_1 + A_2 \cdot x_2}{A_1 + A_2}
\f]

**Геометрический смысл**: координата центра масс двух точек с "массами" \f$ A_1 \f$ и \f$ A_2 \f$.

### 3-точечный (discr3cg)

\f[
x_e = \frac{A_1 \cdot x_1 + A_2 \cdot x_2 + A_3 \cdot x_3}{A_1 + A_2 + A_3}
\f]

### N-точечный (discrncg)

\f[
x_e = \frac{\sum_{i=0}^{N-1} A_i \cdot x_i}{\sum_{i=0}^{N-1} A_i}
\f]

**Защита**: при \f$ \sum A_i = 0 \f$ возвращается средняя точка (ERR-005 fix).

---

## 2. Суммарно-разностный (SD — Sum-Difference)

Классический моноимпульсный метод. Использует отношение разности к сумме.

### 2-точечный (discr2sd)

\f[
x_c = \frac{x_1 + x_2}{2}
\f]

\f[
\Delta x = c \cdot \frac{A_2 - A_1}{A_2 + A_1}
\f]

\f[
x_e = x_c + \Delta x
\f]

где \f$ c \f$ — коэффициент крутизны дискриминаторной характеристики.

**Физический смысл**: \f$ \frac{A_2 - A_1}{A_2 + A_1} \f$ — нормированная разность,
пропорциональная отклонению цели от середины. Коэффициент \f$ c \f$ зависит
от ширины ДН и шага сетки.

---

## 3. Квадратичная аппроксимация (QA — Quadratic Approximation)

Через 3 точки проводится парабола \f$ y = a x^2 + b x + c \f$, находится вершина.

### 3-точечный (discr3qa)

\f[
A_o = \frac{A_2 - A_1}{A_2 - A_3}
\f]

\f[
x_e = \frac{1}{2} \cdot \frac{(A_o - 1) \cdot x_2^2 - A_o \cdot x_3^2 + x_1^2}{(A_o - 1) \cdot x_2 - A_o \cdot x_3 + x_1}
\f]

**Граничные случаи** (ERR-004 fix, используется `DBL_EPSILON`):

| Условие | Результат |
|---------|-----------|
| \f$ A_1 = A_2 = A_3 \f$ | \f$ x_e = x_2 \f$ (центральная точка) |
| \f$ A_2 = A_3 \f$, \f$ A_1 > A_2 \f$ | \f$ x_e = x_1 \f$ |
| \f$ A_2 = A_3 \f$, \f$ A_1 < A_2 \f$ | \f$ x_e = \frac{x_2 + x_3}{2} \f$ |
| \f$ A_1 = A_2 \f$, \f$ A_3 > A_2 \f$ | \f$ x_e = x_3 \f$ |
| \f$ A_1 = A_2 \f$, \f$ A_3 < A_2 \f$ | \f$ x_e = \frac{x_1 + x_2}{2} \f$ |

---

## 4. Экспоненциальная аппроксимация (EA — Exponential Approximation)

Наиболее точный метод. Моделирует ДН функцией Гаусса:

\f[
y(x) = A_{max} \cdot \exp\left( -a \cdot (x - x_0)^2 \right)
\f]

### Алгоритм (discr3ea)

**Шаг 1**: Проверка входных данных
- Все амплитуды должны быть > 0
- Не все амплитуды равны

**Шаг 2**: Сортировка по возрастанию координаты \f$ x \f$

**Шаг 3**: Проверка выпуклости
- Максимум должен быть в центре (а не на краю)
- При вогнутости → `EXIT_FAILURE`

**Шаг 4**: Логарифмирование

\f[
z_i = \ln(A_i), \quad i = 1, 2, 3
\f]

Задача сводится к параболической аппроксимации в логарифмическом масштабе:
\f$ z(x) = -a(x - x_0)^2 + \ln(A_{max}) \f$

**Шаг 5**: Вершина параболы

\f[
\alpha = z_1 (f_2^2 - f_3^2) + z_2 (f_3^2 - f_1^2) + z_3 (f_1^2 - f_2^2)
\f]

\f[
\beta = z_1 (f_2 - f_3) + z_2 (f_3 - f_1) + z_3 (f_1 - f_2)
\f]

\f[
x_e = \frac{\alpha}{2 \beta}
\f]

где \f$ f_i \f$ — координаты после сортировки.

**Шаг 6**: Ограничение на вылет

\f[
f_1 - \frac{f_3 - f_1}{2} \leq x_e \leq f_3 + \frac{f_3 - f_1}{2}
\f]

Оценка не должна выходить за пределы области зондирования более чем на полшага.

### Уточнение амплитуды (discr3eaY)

\f[
a_0 = \frac{z_1 - z_2}{2 x_e (x_1 - x_2) - x_1^2 + x_2^2}
\f]

\f[
y_e = A_2 \cdot \exp\left( a_0 \cdot (x_2 - x_e)^2 \right)
\f]

## 5. МНК-парабола по 5 точкам — 5QA (`discr5qa`)

Идея: вместо параболы, идеально проходящей через **3** точки, аппроксимируем
параболой **5** точек **по методу наименьших квадратов** (МНК / LSQ).
Получаем переопределённую систему — 5 уравнений на 3 неизвестных
\f$ (a, b, c) \f$:

\f[
A_i \approx a\,x_i^2 + b\,x_i + c, \qquad i = 1,\ldots,5
\f]

В матричной форме:
\f[
\mathbf{H}\,\boldsymbol{\theta} = \mathbf{y},
\quad
\mathbf{H} =
\begin{pmatrix}
x_1^2 & x_1 & 1 \\
x_2^2 & x_2 & 1 \\
x_3^2 & x_3 & 1 \\
x_4^2 & x_4 & 1 \\
x_5^2 & x_5 & 1
\end{pmatrix},
\quad
\boldsymbol{\theta} = \begin{pmatrix} a \\ b \\ c \end{pmatrix}.
\f]

Решение МНК (нормальные уравнения):
\f[
(\mathbf{H}^{\!\top}\mathbf{H})\,\boldsymbol{\theta} = \mathbf{H}^{\!\top}\mathbf{y}.
\f]

### Замкнутая формула для равноотстоящих \f$ x_i = \{-2,-1,0,1,2\} \f$

Для равномерной сетки \f$ x_i = -2,-1,0,1,2 \f$ матрица
\f$ \mathbf{H}^{\!\top}\mathbf{H} \f$ блочно-диагональна (так как сумма
нечётных степеней равна нулю), и решение можно записать в **замкнутом виде**:

\f[
\boxed{
a = \frac{2A_1 - A_2 - 2A_3 - A_4 + 2A_5}{14}
}
\f]
\f[
\boxed{
b = \frac{-2A_1 - A_2 + A_4 + 2A_5}{10}
}
\f]
\f[
\boxed{
\delta = -\frac{b}{2a}, \qquad x_e = x_3 + h\,\delta
}
\f]

где \f$ h \f$ — шаг сетки, \f$ x_3 \f$ — центральная точка.

**Свойства:**
- Сложность: O(1) — 6 умножений + 7 сложений + 1 деление
- Не требует никаких циклов и матричных вычислений
- В нормальной зоне даёт ошибку ~2× меньше, чем 3-точечная QA

@image html 02_hanning_vs_parabola.png "Слева — форма пика sinc vs Hanning. Справа — как парабола ложится на Hanning kernel: ошибка <2.5% в пределах ±0.7 бина"

@note Защита: при \f$ |a| < 10^{-30} \f$ возвращается \f$ x_e = x_3 \f$
(данные плоские — оценить нельзя).

## 6. МНК-Гауссиан по 5 точкам — 5EA (`discr5ea`)

Самый точный из реализованных методов. Аппроксимирует пик **гауссианом**
(колоколом) методом наименьших квадратов:

\f[
y(x) \approx A_{\max} \cdot \exp\!\bigl(-a\,(x - x_0)^2\bigr)
\f]

### Алгоритм

**Шаг 1.** Логарифмирование амплитуд:
\f[
z_i = \ln A_i, \quad i = 1,\ldots,5
\f]

В лог-пространстве гауссиан превращается в **параболу**:
\f[
z(x) = -a\,(x - x_0)^2 + \ln A_{\max} = a'x^2 + b'x + c'.
\f]

**Шаг 2.** К `z[]` применяем те же замкнутые МНК-формулы, что и в 5QA:
\f[
a = \frac{2z_1 - z_2 - 2z_3 - z_4 + 2z_5}{14}
\f]
\f[
b = \frac{-2z_1 - z_2 + z_4 + 2z_5}{10}
\f]

**Шаг 3.** Вершина параболы → оценка \f$ x_e \f$:
\f[
\delta = -\frac{b}{2a}, \qquad x_e = x_3 + h\,\delta.
\f]

### Когда не работает (и что делать)

Метод требует **строго положительных** амплитуд: \f$ A_i > 0 \f$.
Иначе \f$ \ln A_i \f$ не определён.

**Стратегия fallback** в боевом коде:
```c
int ok = discr5ea(A, x, &xe);
if (ok != 0) {
  // что-то ≤ 0 или результат за границей
  discr5qa(A, x, &xe);   // ← переходим на 5QA по амплитудам
}
```

### Почему именно Гауссиан

Главное наблюдение проекта: после применения **окна Hanning** и FFT
форма одиночного спектрального пика (Hanning kernel) **очень близка к
гауссиану** в пределах главного лепестка. Поэтому модель
\f$ y \approx A_{\max}\exp(-a(x-x_0)^2) \f$ является **физически
обоснованной** для типичного FFT-пайплайна.

## 8. Обёртка discr3() — коэффициент пересчёта

Для обёрток `discr3()` и `discr3_()` координаты пересчитываются:

\f[
c = \frac{2 \pi \cdot dx}{\lambda}
\f]

где \f$ dx \f$ — шаг антенной решётки, \f$ \lambda \f$ — длина волны.

Координаты масштабируются: \f$ x' = x \cdot c \f$, оценка возвращается в исходных единицах: \f$ x_e = x'_e / c \f$.

@note После рефакторинга (ERR-003 fix) входные массивы **не модифицируются** —
координаты умножаются на \f$ c \f$ при передаче в дискриминатор, а не в массиве.
