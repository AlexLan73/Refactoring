## Дискриминаторные оценки координат (КППО / ТЗК)

> **Модуль**: discriminator_estimates
> **Организация**: ОАО НПК НИИДАР

> **Рефакторинг**: Кодо (AI-ассистент), 2026-03-28

---

### Назначение

Библиотека реализует **дискриминаторные оценки координат цели** по отсчётам
диаграммы направленности (ДН) антенны. Применяется в комплексе программ
первичной обработки (КППО) для вычисления текущих замеров координат (ТЗК).

### Физический смысл

Диаграмма направленности антенны описывается функцией, близкой к:

\f[
\text{sinc}(x) = \frac{\sin(x)}{x}
\f]

Антенна "смотрит" в несколько направлений (2-3 луча) и измеряет амплитуду
принятого сигнала в каждом. По этим отсчётам дискриминатор оценивает
**истинное положение цели** между лучами.

### 4 метода аппроксимации

| Метод | Код | Точность | Скорость | Описание |
|-------|-----|----------|----------|----------|
| Центр тяжести | `DT_CG` | ±0.3 шага | Максимальная | Взвешенное среднее координат |
| Суммарно-разностный | `DT_SD` | ±0.2 шага | Максимальная | Классический моноимпульсный |
| Квадратичная | `DT_QA` | ±0.1 шага | Высокая | Парабола через 3 точки |
| Экспоненциальная | `DT_EA` | ±0.05 шага | Средняя | Гауссова кривая через 3 точки |

### Быстрый старт

**C (3-точечный QA дискриминатор):**
```c
#include "discrqa.h"

double A1 = 0.75, A2 = 2.75, A3 = 2.75;  // амплитуды ДН
double x1 = -1.0, x2 = 0.0, x3 = 1.0;    // координаты лучей
double xe = discr3qa(A1, A2, A3, x1, x2, x3);
// xe ≈ 0.5 — оценка координаты цели
```

### Зависимость ошибки от шага сетки

@image html error_vs_grid_step.png "Рис. 3: Ошибка оценки при разных шагах сетки"

При мелкой сетке (шаг < 0.5) все методы дают хорошую точность.
При крупной сетке (шаг > 1.5) экспоненциальный метод (EA) значительно
точнее квадратичного (QA) и центра масс (CG).
