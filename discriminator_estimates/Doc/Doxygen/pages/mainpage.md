## Назначение

Библиотека реализует **дискриминаторные оценки координат цели** по отсчётам
диаграммы направленности (ДН) антенны. Применяется в КППО для ТЗК.

## Физический смысл

Диаграмма направленности антенны описывается функцией, близкой к:

\f[
\text{sinc}(x) = \frac{\sin(x)}{x}
\f]

Антенна "смотрит" в несколько направлений (2-3 луча) и измеряет амплитуду.
По отсчётам дискриминатор оценивает **истинное положение цели** между лучами.

## 4 метода оценки координат

| Метод | Код | Точность | Файл | Описание |
|-------|-----|----------|------|----------|
| Центр тяжести | `DT_CG` | ±0.3 шага | discrcg.c | Взвешенное среднее |
| Суммарно-разностный | `DT_SD` | ±0.2 шага | discrsd.c | Моноимпульсный |
| Квадратичная | `DT_QA` | ±0.1 шага | discrqa.c | Парабола через 3 точки |
| Экспоненциальная | `DT_EA` | ±0.05 шага | discrea.c | Гауссова кривая |

## FFT-применение: оценка частоты по спектру

Те же дискриминаторы (CG, SD, QA/SQR, EA/EXP) применяются для **оценки частоты
по FFT-спектру** комплексного сигнала. Дополнительно реализован метод Jacobsen (LAY).

## Быстрый старт

**C (3-точечный QA дискриминатор):**
```c
#include "discrqa.h"

double A1 = 0.75, A2 = 2.75, A3 = 2.75;
double x1 = -1.0, x2 = 0.0, x3 = 1.0;
double xe = discr3qa(A1, A2, A3, x1, x2, x3);
// xe ~ 0.5
```
