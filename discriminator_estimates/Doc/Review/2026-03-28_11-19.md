## Почему sinc(x)

Функция аппроксимируемая дискриминаторами близка к **sinc(x) = sin(x)/x** — это:
- Диаграмма направленности (ДН) равномерной линейной антенны
- Стандартная модель в радиолокации для оценки координат цели
- Имеет один главный максимум и боковые лепестки

Экспериментальные данные в тестах — отсчёты sinc(x - x0) на регулярной сетке,
где x0 — истинное смещение пика (от 0 до 0.45 шага сетки).

## Как собрать и запустить

### GTest (C++):
```bash
cd discriminator_estimates
mkdir build && cd build
cmake .. -G "MinGW Makefiles"  # или Ninja, или VS
cmake --build .
./test_discr_gtest      # запуск всех gtest
ctest                   # или через CTest
```

### Python тесты:
```bash
python test_python/test_discriminators.py      # тесты
python test_python/test_discriminators_plot.py  # графики
```
