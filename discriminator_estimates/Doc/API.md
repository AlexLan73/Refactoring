## Типы данных

### DISCRTYPE

Определен в `discr_types.h`.

```c
typedef enum {
  DT_NO = 1,   // Без уточнения
  DT_CG = 2,   // Центр масс (Center of Gravity)
  DT_SD = 3,   // Суммарно-разностный (Sum-Difference)
  DT_QA = 4,   // Квадратичная аппроксимация (Quadratic Approximation)
  DT_EA = 5,   // Экспоненциальная аппроксимация (Exponential Approximation)
} DISCRTYPE;
```

## CG -- Центр тяжести

Заголовок: `discrcg.h` | Реализация: `discrcg.c`

### discr2cg

Двухточечный дискриминатор, центр тяжести.

```c
double discr2cg(double A1, double A2, double x1, double x2);
```

**Параметры:**

| Имя | Тип | Описание |
|-----|-----|----------|
| `A1` | `double` | Амплитуда первой точки |
| `A2` | `double` | Амплитуда второй точки |
| `x1` | `double` | Координата первой точки |
| `x2` | `double` | Координата второй точки |

**Возвращает:** `double` -- оценка координаты.

**Формула:**

```
xe = (A1*x1 + A2*x2) / (A1 + A2)
```

**Граничные случаи:**

- `A1 + A2 = 0` -- возвращает середину `(x1 + x2) / 2`.

**Пример:**

```c
#include "discrcg.h"

double x = discr2cg(0.8, 1.0, -1.0, 1.0);
// x ~ 0.111 (сдвиг в сторону большей амплитуды)
```

### discr3cg

Трёхточечный дискриминатор, центр тяжести.

```c
double discr3cg(double A1, double A2, double A3,
                double x1, double x2, double x3);
```

**Параметры:**

| Имя | Тип | Описание |
|-----|-----|----------|
| `A1` | `double` | Амплитуда первой точки |
| `A2` | `double` | Амплитуда второй точки |
| `A3` | `double` | Амплитуда третьей точки |
| `x1` | `double` | Координата первой точки |
| `x2` | `double` | Координата второй точки |
| `x3` | `double` | Координата третьей точки |

**Возвращает:** `double` -- оценка координаты.

**Формула:**

```
xe = (A1*x1 + A2*x2 + A3*x3) / (A1 + A2 + A3)
```

**Граничные случаи:**

- `A1 + A2 + A3 = 0` -- возвращает центральную точку `x2`.

**Пример:**

```c
#include "discrcg.h"

double x = discr3cg(0.5, 1.0, 0.5, -1.0, 0.0, 1.0);
// x = 0.0 (симметричная ДН, максимум в центре)
```

---

### discrncg

Дискриминатор для произвольного числа точек, центр тяжести.

```c
double discrncg(const double *A, const double *x, unsigned int N);
```

**Параметры:**

| Имя | Тип | Описание |
|-----|-----|----------|
| `A` | `const double*` | Массив амплитуд (N элементов) |
| `x` | `const double*` | Массив координат (N элементов) |
| `N` | `unsigned int` | Число точек |

**Возвращает:** `double` -- оценка координаты.

**Формула:**

```
xe = sum(A[i]*x[i], i=0..N-1) / sum(A[i], i=0..N-1)
```

**Граничные случаи:**

- `sum(A) = 0` -- возвращает `x[N/2]` (центральный элемент).
- `N = 0` -- возвращает `0.0`.

**Пример:**

```c
#include "discrcg.h"

double A[] = {0.3, 0.8, 1.0, 0.7, 0.2};
double x[] = {-2.0, -1.0, 0.0, 1.0, 2.0};
double xe = discrncg(A, x, 5);
// xe ~ -0.067 (взвешенный центр масс)
```

## SD -- Суммарно-разностный

Заголовок: `discrsd.h` | Реализация: `discrsd.c`

### discr2sd

Двухточечный дискриминатор, суммарно-разностный.

```c
double discr2sd(double c, double A1, double A2,
                double x1, double x2);
```

**Параметры:**

| Имя | Тип | Описание |
|-----|-----|----------|
| `c` | `double` | Коэффициент суммарно-разностного дискриминатора |
| `A1` | `double` | Амплитуда первой точки |
| `A2` | `double` | Амплитуда второй точки |
| `x1` | `double` | Координата первой точки |
| `x2` | `double` | Координата второй точки |

**Возвращает:** `double` -- оценка координаты.

**Формула:**

```
xc = (x1 + x2) / 2
dx = c * (A2 - A1) / (A2 + A1)
xe = xc + dx
```

**Граничные случаи:**

- `A1 + A2 = 0` -- возвращает середину `(x1 + x2) / 2`.

**Пример:**

```c
#include "discrsd.h"

// c = 2*pi*dx_angle/lambda
double c = 0.5;
double x = discr2sd(c, 0.8, 1.0, -1.0, 1.0);
// x ~ 0.056
```

## QA -- Квадратичная аппроксимация

Заголовок: `discrqa.h` | Реализация: `discrqa.c`

### discr3qa

Трёхточечный дискриминатор, аппроксимация квадратичной зависимостью (вершина параболы).

```c
double discr3qa(double A1, double A2, double A3,
                double x1, double x2, double x3);
```

**Параметры:**

| Имя | Тип | Описание |
|-----|-----|----------|
| `A1` | `double` | Амплитуда первой точки |
| `A2` | `double` | Амплитуда второй точки |
| `A3` | `double` | Амплитуда третьей точки |
| `x1` | `double` | Координата первой точки |
| `x2` | `double` | Координата второй точки |
| `x3` | `double` | Координата третьей точки |

**Возвращает:** `double` -- оценка координаты (вершина аппроксимирующей параболы).

**Формула:**

```
Ao = (A2 - A1) / (A2 - A3)
xe = 0.5 * ((Ao-1)*x2^2 - Ao*x3^2 + x1^2) / ((Ao-1)*x2 - Ao*x3 + x1)
```

Аппроксимирует 3 точки параболой `y = a*x^2 + b*x + c` и находит координату вершины.

**Граничные случаи:**

| Условие | Результат |
|---------|-----------|
| `A2 = A3` и `A1 = A2` | `x2` (все равны) |
| `A2 = A3` и `A1 > A2` | `x1` |
| `A2 = A3` и `A1 < A2` | `(x2 + x3) / 2` |
| `A1 = A2` и `A3 > A2` | `x3` |
| `A1 = A2` и `A3 <= A2` | `(x1 + x2) / 2` |
| Знаменатель = 0 | `x2` |

**Пример:**

```c
#include "discrqa.h"

// sinc-подобная ДН, сдвиг 0.3
double x = discr3qa(0.858, 1.0, 0.947, -1.0, 0.0, 1.0);
// x ~ -0.294 (оценка смещения)
```

## EA -- Экспоненциальная аппроксимация

Заголовок: `discrea.h` | Реализация: `discrea.c`

### discr3ea

Трёхточечный дискриминатор, аппроксимация экспоненциальной зависимостью.

Аппроксимирует ДН функцией `y = A_max * exp(-a * (x - x0)^2)` и находит координату максимума `x0`.

```c
int discr3ea(double A1, double A2, double A3,
             double x1, double x2, double x3, double *xe);
```

**Параметры:**

| Имя | Тип | Описание |
|-----|-----|----------|
| `A1` | `double` | Амплитуда первой точки (должна быть > 0) |
| `A2` | `double` | Амплитуда второй точки (должна быть > 0) |
| `A3` | `double` | Амплитуда третьей точки (должна быть > 0) |
| `x1` | `double` | Координата первой точки |
| `x2` | `double` | Координата второй точки |
| `x3` | `double` | Координата третьей точки |
| `xe` | `double*` | [out] Оценка координаты |

**Возвращает:** `int` -- `EXIT_SUCCESS` (0) при успехе, `EXIT_FAILURE` (1) при ошибке. При ошибке `*xe` содержит приближенное значение.

**Алгоритм:**

1. Проверка: все амплитуды > 0, не все равны.
2. Сортировка по координатам (x по возрастанию).
3. Проверка на выпуклость (максимум должен быть в центре).
4. Логарифмирование: `z = ln(A)`, аппроксимация параболой `z(x)`.
5. Вершина параболы: `xe = 0.5 * a / b`.
6. Ограничение на вылет за пределы зондирования.

**Граничные случаи / коды ошибок (EXIT_FAILURE):**

| Условие | `*xe` содержит |
|---------|----------------|
| Любая амплитуда <= 0 | `x2` |
| Все амплитуды равны | `x2` |
| Нет выпуклости (максимум на краю) | `x1` или `x3` |
| Знаменатель `b = 0` | Экстраполированная координата |
| Результат за пределами зондирования | Ограниченная координата |

**Пример:**

```c
#include "discrea.h"
#include <stdlib.h>

double xe;
int rc = discr3ea(0.858, 1.0, 0.947, -1.0, 0.0, 1.0, &xe);
if (rc == EXIT_SUCCESS) {
    printf("Координата: %.4f\n", xe);
} else {
    printf("Ошибка, приближение: %.4f\n", xe);
}
```

### discr3eaY

Уточнение амплитуды по экспоненциальной аппроксимации.

Используется после `discr3ea()` для вычисления амплитуды в найденной точке `xe`.

```c
double discr3eaY(double A1, double A2, double x1, double x2, double xe);
```

**Параметры:**

| Имя | Тип | Описание |
|-----|-----|----------|
| `A1` | `double` | Амплитуда первой точки |
| `A2` | `double` | Амплитуда второй точки |
| `x1` | `double` | Координата первой точки |
| `x2` | `double` | Координата второй точки |
| `xe` | `double` | Оценка координаты (результат `discr3ea`) |

**Возвращает:** `double` -- уточненное значение амплитуды в точке `xe`.

**Формула:**

```
a0 = (ln(A1) - ln(A2)) / (2*xe*(x1 - x2) - x1^2 + x2^2)
ye = A2 * exp(a0 * (x2 - xe)^2)
```

**Граничные случаи:**

- `A1 <= 0` или `A2 <= 0` -- возвращает `0.0`.
- Знаменатель для `a0` равен 0 -- возвращает `max(A1, A2)`.

**Пример:**

```c
#include "discrea.h"

double xe;
discr3ea(0.858, 1.0, 0.947, -1.0, 0.0, 1.0, &xe);
double amp = discr3eaY(0.858, 1.0, -1.0, 0.0, xe);
printf("Амплитуда в xe=%.3f: %.4f\n", xe, amp);
```

## Обёртки

Заголовок: `discr_common.h` | Реализация: `discr_common.c`

Высокоуровневые функции для работы с матрицами координат ДН (диаграмм направленности).

### discr2

Двухточечный дискриминатор для матриц 2x2 (обертка над `discr2cg`).

```c
int discr2(double maz[2][2], double mel[2][2], double mval[2][2],
           double *az, double *el, double *val);
```

**Параметры:**

| Имя | Тип | Описание |
|-----|-----|----------|
| `maz` | `double[2][2]` | Матрица азимутальных координат |
| `mel` | `double[2][2]` | Матрица угломестных координат |
| `mval` | `double[2][2]` | Матрица амплитуд ДН |
| `az` | `double*` | [out] Уточненная координата азимута |
| `el` | `double*` | [out] Уточненная координата угла места |
| `val` | `double*` | [out] Максимальное значение амплитуды |

**Возвращает:** `int` -- `0` при успехе, `-1` при ошибке (если какая-либо амплитуда <= 0).

**Алгоритм:**

1. Проверка: все амплитуды > 0.
2. Поиск строки/столбца с максимальной амплитудой.
3. Оценка азимута: `discr2cg` по строке с максимумом.
4. Оценка угла места: `discr2cg` по столбцу с максимумом.

**Пример:**

```c
#include "discr_common.h"

double maz[2][2]  = {{ -0.5, 0.5 }, { -0.5, 0.5 }};
double mel[2][2]  = {{ -0.5, -0.5 }, { 0.5, 0.5 }};
double mval[2][2] = {{ 0.7, 0.9 }, { 0.8, 1.0 }};
double az, el, val;

int rc = discr2(maz, mel, mval, &az, &el, &val);
// rc = 0, az ~ 0.263, el ~ 0.056, val = 1.0
```

### discr3

Трёхточечный дискриминатор для динамических матриц (обёртка, поддерживает DT_CG / DT_QA / DT_EA).

```c
int discr3(double **maz, double **mel, double **mval,
           int ndnx, int ndny,
           DISCRTYPE discrtype, double dx, double lambda,
           double *az, double *el, double *val);
```

**Параметры:**

| Имя | Тип | Описание |
|-----|-----|----------|
| `maz` | `double**` | Массив азимутальных координат [ndnx][ndny] |
| `mel` | `double**` | Массив угломестных координат [ndnx][ndny] |
| `mval` | `double**` | Массив амплитуд [ndnx][ndny] |
| `ndnx` | `int` | Число ДН по оси X (строки) |
| `ndny` | `int` | Число ДН по оси Y (столбцы) |
| `discrtype` | `DISCRTYPE` | Тип дискриминатора: `DT_CG`, `DT_QA`, `DT_EA` |
| `dx` | `double` | Шаг по углу (для DT_QA, DT_EA; масштаб координат) |
| `lambda` | `double` | Длина волны (для DT_QA, DT_EA; масштаб координат) |
| `az` | `double*` | [out] Уточненная координата азимута |
| `el` | `double*` | [out] Уточненная координата угла места |
| `val` | `double*` | [out] Уточненное значение амплитуды |

**Возвращает:** `int` --
- `0` -- успех;
- `-1` -- нельзя использовать 3-точечный дискриминатор (амплитуда <= 0);
- `-2` -- неизвестный тип дискриминатора.

**Масштабирование координат (DT_QA, DT_EA):**

Для DT_QA и DT_EA координаты масштабируются на `c = 2*pi*dx/lambda` перед вычислением и обратно после. DT_CG работает без масштабирования.

**Пример:**

```c
#include "discr_common.h"
#include <stdlib.h>

// Выделение памяти под 3x3 матрицы
double *maz_data[]  = { /* ... указатели на строки ... */ };
double *mel_data[]  = { /* ... */ };
double *mval_data[] = { /* ... */ };

double az, el, val;
double dx = 0.01;       // шаг по углу (рад)
double lambda = 0.03;   // длина волны (м)

int rc = discr3(maz_data, mel_data, mval_data, 3, 3,
                DT_QA, dx, lambda,
                &az, &el, &val);
```

### discr3_

Адаптер `discr3` для статических массивов `double[3][3]`.

Преобразует массивы `double[3][3]` в `double**` и вызывает `discr3()`.

```c
int discr3_(double maz[3][3], double mel[3][3], double mval[3][3],
            DISCRTYPE discrtype, double dx, double lambda,
            double *az, double *el, double *val);
```

**Параметры:**

| Имя | Тип | Описание |
|-----|-----|----------|
| `maz` | `double[3][3]` | Матрица азимутальных координат |
| `mel` | `double[3][3]` | Матрица угломестных координат |
| `mval` | `double[3][3]` | Матрица амплитуд ДН |
| `discrtype` | `DISCRTYPE` | Тип дискриминатора |
| `dx` | `double` | Шаг по углу |
| `lambda` | `double` | Длина волны |
| `az` | `double*` | [out] Уточненная координата азимута |
| `el` | `double*` | [out] Уточненная координата угла места |
| `val` | `double*` | [out] Уточненное значение амплитуды |

**Возвращает:** `int` -- то же, что `discr3()` (0, -1, -2).

**Пример:**

```c
#include "discr_common.h"

double maz[3][3]  = {{ -1, 0, 1 }, { -1, 0, 1 }, { -1, 0, 1 }};
double mel[3][3]  = {{ -1, -1, -1 }, { 0, 0, 0 }, { 1, 1, 1 }};
double mval[3][3] = {
    { 0.5, 0.7, 0.5 },
    { 0.7, 1.0, 0.7 },
    { 0.5, 0.7, 0.5 }
};
double az, el, val;

int rc = discr3_(maz, mel, mval, DT_QA, 0.01, 0.03, &az, &el, &val);
```

### checkuse3d

Проверка возможности использования 3-точечного дискриминатора.

Проверяет, что все амплитуды в матрице положительны (> `DBL_EPSILON`).

```c
int checkuse3d(double **maz, double **mel, double **mval,
               int ndnx, int ndny);
```

**Параметры:**

| Имя | Тип | Описание |
|-----|-----|----------|
| `maz` | `double**` | Массив азимутальных координат (не используется, для совместимости) |
| `mel` | `double**` | Массив угломестных координат (не используется, для совместимости) |
| `mval` | `double**` | Массив амплитуд |
| `ndnx` | `int` | Число строк |
| `ndny` | `int` | Число столбцов |

**Возвращает:** `int` -- `1` -- можно использовать, `0` -- нельзя (есть нулевые амплитуды).

**Пример:**

```c
#include "discr_common.h"

if (checkuse3d(maz, mel, mval, 3, 3)) {
    // Можно вызывать discr3()
    discr3(maz, mel, mval, 3, 3, DT_QA, dx, lambda, &az, &el, &val);
} else {
    // Перейти на 2-точечный: conv3to2 + discr2
}
```

### conv3to2

Извлечение подматрицы 2x2 из 3x3 с максимальной суммарной амплитудой.

Используется для перехода с 3-точечного на 2-точечный дискриминатор, когда `checkuse3d` возвращает 0.

```c
int conv3to2(double maz3[3][3], double mel3[3][3], double mval3[3][3],
             double maz2[2][2], double mel2[2][2], double mval2[2][2]);
```

**Параметры:**

| Имя | Тип | Описание |
|-----|-----|----------|
| `maz3` | `double[3][3]` | [in] Азимутальные координаты 3x3 |
| `mel3` | `double[3][3]` | [in] Угломестные координаты 3x3 |
| `mval3` | `double[3][3]` | [in] Амплитуды 3x3 |
| `maz2` | `double[2][2]` | [out] Азимутальные координаты 2x2 |
| `mel2` | `double[2][2]` | [out] Угломестные координаты 2x2 |
| `mval2` | `double[2][2]` | [out] Амплитуды 2x2 |

**Возвращает:** `int` -- `0` (всегда успех).

**Алгоритм:** Перебираются все 4 квадранта 2x2 внутри матрицы 3x3 (смещения `(0,0)`, `(0,1)`, `(1,0)`, `(1,1)`). Выбирается квадрант с максимальной суммой амплитуд.

**Пример:**

```c
#include "discr_common.h"

double maz3[3][3]  = {{ -1, 0, 1 }, { -1, 0, 1 }, { -1, 0, 1 }};
double mel3[3][3]  = {{ -1, -1, -1 }, { 0, 0, 0 }, { 1, 1, 1 }};
double mval3[3][3] = {
    { 0.0, 0.7, 0.5 },    // mval3[0][0] = 0 -- нельзя 3-точечный
    { 0.7, 1.0, 0.8 },
    { 0.5, 0.8, 0.6 }
};

double maz2[2][2], mel2[2][2], mval2[2][2];
conv3to2(maz3, mel3, mval3, maz2, mel2, mval2);
// Выбран квадрант (0,1)-(1,2) с максимальной суммой

double az, el, val;
discr2(maz2, mel2, mval2, &az, &el, &val);
```

## МНК 5-точечные

Заголовок: `discr5ea.h` | Реализация: `discr5ea.c`

МНК-дискриминаторы по 5 равноотстоящим отсчётам. Используют метод наименьших квадратов для аппроксимации переопределённой системы (5 уравнений, 3 неизвестных). Более устойчивы к шуму, чем 3-точечные методы.

### discr5ea

Пятиточечный дискриминатор, МНК-гауссова аппроксимация (LSQ Gaussian 5pt).

```c
int discr5ea(const double A[5], const double x[5], double *xe);
```

**Параметры:**

| Имя | Тип | Описание |
|-----|-----|----------|
| `A` | `const double[5]` | Массив амплитуд [A1..A5], центральная = A[2] |
| `x` | `const double[5]` | Массив координат (равноотстоящие) |
| `xe` | `double*` | [out] Оценка координаты |

**Возвращает:** `int` -- `EXIT_SUCCESS` (0) при успехе, `EXIT_FAILURE` (1) при ошибке.

**Алгоритм:** Логарифмирует амплитуды (`z = ln(A)`), затем МНК-парабола по нормированным координатам `{-2, -1, 0, 1, 2}`:

```
a = (2z1 - z2 - 2z3 - z4 + 2z5) / 14
b = (-2z1 - z2 + z4 + 2z5) / 10
xe = x3 + h * (-b / (2*a))
```

**Условие ошибки:** любая `A[i] <= 0`, нулевой шаг сетки, или парабола не вогнутая (`a >= 0`).

**Пример:**

```c
#include "discr5ea.h"

double A[5] = {0.5, 0.85, 1.0, 0.90, 0.6};
double x[5] = {-2.0, -1.0, 0.0, 1.0, 2.0};
double xe;
int rc = discr5ea(A, x, &xe);
// rc = EXIT_SUCCESS, xe ~ 0.0
```

---

### discr5qa

Пятиточечный дискриминатор, МНК-параболическая аппроксимация (LSQ Parabolic 5pt).

```c
int discr5qa(const double A[5], const double x[5], double *xe);
```

**Параметры:** аналогично `discr5ea`.

**Возвращает:** `int` -- `EXIT_SUCCESS` (0) при успехе, `EXIT_FAILURE` (1) при ошибке.

**Алгоритм:** Как `discr5ea`, но аппроксимирует непосредственно амплитуды (без логарифмирования):

```
a = (2A1 - A2 - 2A3 - A4 + 2A5) / 14
b = (-2A1 - A2 + A4 + 2A5) / 10
xe = x3 + h * (-b / (2*a))
```

**Пример:**

```c
#include "discr5ea.h"

double A[5] = {0.5, 0.85, 1.0, 0.90, 0.6};
double x[5] = {-2.0, -1.0, 0.0, 1.0, 2.0};
double xe;
int rc = discr5qa(A, x, &xe);
// rc = EXIT_SUCCESS, xe ~ 0.0 (менее точно чем discr5ea для гауссоподобных ДН)
```

## Автоматический дискриминатор

Заголовок: `discr_auto.h` | Реализация: `discr_auto.c`

Решает проблему **монотонных данных** — когда пик ДН находится за пределами окна из 3 отсчётов (67% реальных случаев). В этом режиме стандартные 3-точечные методы (EA, QA) дают ошибки > 25% шага сетки.

### discr_is_monotonic

Проверка монотонности трёх отсчётов.

```c
int discr_is_monotonic(double A1, double A2, double A3);
```

**Параметры:** `A1`, `A2`, `A3` -- амплитуды трёх точек.

**Возвращает:** `int` -- `1` если монотонные (пик за сеткой), `0` если есть перегиб (пик внутри).

**Пример:**

```c
#include "discr_auto.h"

// Монотонные: пик правее сетки
if (discr_is_monotonic(2.1, 4.5, 7.8)) {
    printf("Пик за пределами сетки!\n");
}
```

---

### discr3_extrap_gauss

Гауссова экстраполяция (E2) для монотонных случаев.

```c
int discr3_extrap_gauss(double A1, double A2, double A3,
                        double x1, double x2, double x3,
                        double *xe);
```

**Параметры:**

| Имя | Тип | Описание |
|-----|-----|----------|
| `A1`, `A2`, `A3` | `double` | Амплитуды (должны быть > 0) |
| `x1`, `x2`, `x3` | `double` | Координаты точек |
| `xe` | `double*` | [out] Экстраполированная оценка |

**Возвращает:** `int` -- `0` при успехе, `1` при ошибке (нулевые амплитуды).

**Алгоритм:** Как `discr3ea`, но без проверки выпуклости и с расширенным допуском вылета (2× вместо 0.5× от размера сетки). MAE = 0.148 для монотонных случаев (в 2.2 раза лучше параболической экстраполяции).

### discr3_auto

Автоматический дискриминатор с переключением.

```c
int discr3_auto(double A1, double A2, double A3,
                double x1, double x2, double x3,
                double *xe);
```

**Параметры:**

| Имя | Тип | Описание |
|-----|-----|----------|
| `A1`, `A2`, `A3` | `double` | Амплитуды трёх точек |
| `x1`, `x2`, `x3` | `double` | Координаты точек |
| `xe` | `double*` | [out] Оценка координаты |

**Возвращает:** `int` -- код режима:
- `0` -- EA успех (нормальный режим, лучший результат)
- `1` -- использован QA как fallback (EA не сошёлся)
- `2` -- использована гауссова экстраполяция E2 (монотонный режим)
- `3` -- E2 не сошлась, возвращает `x2`

**Алгоритм выбора:**

```
if (is_monotonic):
    xe = extrap_gauss(...)    // E2 — гауссова экстраполяция
else:
    ret = discr3ea(...)       // EA — лучший метод
    if (ret == FAILURE):
        xe = discr3qa(...)    // QA — fallback
```

**Пример:**

```c
#include "discr_auto.h"

double xe;
int mode = discr3_auto(2.1, 4.5, 7.8, -1.0, 0.0, 1.0, &xe);
if (mode == 2) {
    printf("Монотонный режим, экстраполяция: xe = %.4f\n", xe);
} else if (mode == 0) {
    printf("EA успех: xe = %.4f\n", xe);
}
```

## Типовой сценарий использования

```
                checkuse3d
                   |
           +-------+-------+
           | (1)           | (0)
       discr3/discr3_   conv3to2
           |               |
       (az, el, val)    discr2
                           |
                       (az, el, val)
```

1. Вызвать `checkuse3d()` -- можно ли использовать 3-точечный.
2. Если да -- вызвать `discr3()` или `discr3_()` с нужным типом.
3. Если нет -- вызвать `conv3to2()` для извлечения лучшего квадранта 2x2, затем `discr2()`.

### Загрузка C-библиотеки (ctypes)

```python
from test_python.analysis.common import load_discr_lib
import ctypes

lib = load_discr_lib()

# discr3qa (возвращает double)
lib.discr3qa.restype  = ctypes.c_double
lib.discr3qa.argtypes = [ctypes.c_double] * 6
xe = lib.discr3qa(A1, A2, A3, x1, x2, x3)

# discr5ea (возвращает int, xe через указатель)
lib.discr5ea.restype  = ctypes.c_int
lib.discr5ea.argtypes = [
    ctypes.POINTER(ctypes.c_double),  # A[5]
    ctypes.POINTER(ctypes.c_double),  # x[5]
    ctypes.POINTER(ctypes.c_double),  # xe
]
A_arr = (ctypes.c_double * 5)(*A_list)
x_arr = (ctypes.c_double * 5)(*x_list)
xe_out = ctypes.c_double(0.0)
rc = lib.discr5ea(A_arr, x_arr, ctypes.byref(xe_out))
```
